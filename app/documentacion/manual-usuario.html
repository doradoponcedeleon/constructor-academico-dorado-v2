Inicio rápido

1. Abre la plataforma desde el navegador.
2. En el panel lateral, entra a Motor de ideas y crea un documento base.
3. Presiona Aplicar al editor para cargarlo en el Editor.
4. Ajusta secciones en Editor.
5. Exporta en Exportadores o genera un paper/tesis.

Navegación general

La plataforma tiene un panel lateral con módulos. Cada botón abre un panel en el área principal.

Editor

Gestiona secciones académicas y contenido.

Motor de ideas

Genera un documento base en Markdown.

Motor Paper / Tesis

Genera estructuras académicas avanzadas.

Motor de Ideas

Convierte ideas en un documento base estructurado.

1. Ingresa Problema, Ideas, Conceptos y Objetivos.
2. Presiona Generar documento base.
3. Revisa el preview (Markdown).
4. Pulsa Aplicar al editor para transferirlo.

El documento se guarda en localStorage.documento_base.

Editor Académico

Gestiona secciones del documento académico.

1. Abre Editor desde el panel lateral.
2. Edita títulos y contenido.
3. Agrega o elimina secciones.
4. Usa Guardar editor para persistencia.

Motor Paper

Genera un paper académico a partir de documento_base.

1. Escribe Título, Autores y Área académica.
2. Pulsa Generar paper.
3. Revisa el preview.
4. Usa Enviar al editor.

Referencias APA

Guarda referencias y genera formato APA.

1. Ingresa Autor(es), Año, Título, Revista/Libro, Editorial y DOI/URL.
2. Pulsa Agregar referencia.
3. Usa Insertar en documento para agregarlas al final del Editor.

Buscador Papers

Búsqueda simulada con resultados realistas.

1. Escribe tema y opcionalmente autor/año.
2. Presiona Buscar papers.
3. Usa Agregar a referencias en un resultado.

Citas APA

Genera citas APA rápidas.

1. Completa el formulario de cita.
2. Presiona Generar cita APA.
3. Guarda en referencias si lo necesitas.

Asistente de Redacción

Mejora o transforma texto académico.

1. Selecciona una sección.
2. Usa las opciones: Mejorar, Resumir, Ampliar, Formalizar.
3. Presiona Aplicar al editor.

Generador de Tesis

Crea una estructura completa de tesis con datos de base.

1. Completa Título, Autor, Carrera, Universidad y Línea.
2. Pulsa Generar tesis.
3. Usa Enviar al editor para transferirla.

Exportadores

Exporta el documento en varios formatos.

1. Entra a Exportadores.
2. Elige PDF, DOCX, HTML, TXT o JSON.
3. Descarga el archivo generado.

Buenas prácticas

  • Guarda referencias antes de exportar.
  • Revisa el preview antes de aplicar al editor.
  • Actualiza secciones de forma progresiva.

Solución de problemas

  • No se ve un módulo: revisa la navegación lateral.
  • No se carga el preview: revisa que el contenido exista en localStorage.
  • Exportación PDF falla: confirma que jsPDF está cargado.
